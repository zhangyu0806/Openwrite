# 通用节奏控制技法 (Universal Rhythm Craft)

> 只保留跨作品可复用的节奏规律，不保留具体作品对照和签名式写法。

## 一、段落长度控制

### 基本建议

| 段落类型 | 常见用途 |
|----------|----------|
| 短段 | 反应、转折、危险、吐槽 |
| 中段 | 日常推进、说明、交流 |
| 长段 | 策略推演、环境铺排、情感蓄力 |

### 核心原则

- 长短参差，不要连续 3 个以上同长度段落
- 短段要敢用，它能提供呼吸点和重音
- 长段要有理由，不要为了“像文学”而拖长

## 二、紧张-松弛循环

### 基本循环

```text
铺垫 → 紧张递升 → 高潮/反转 → 消化 → 新铺垫
```

### 常见层级

| 层级 | 尺度 |
|------|------|
| 微循环 | 几段内部的起伏 |
| 小循环 | 一章内场景起伏 |
| 中循环 | 数章组成的事件起伏 |
| 大循环 | 一节或一篇的整体情绪波形 |

### 禁忌

- 连续多章同一紧张度
- 高潮之后立刻接下一个高潮
- 日常太久没有前进信号

## 三、加速技巧

- 连续短段
- 对话标签减量
- 直接动作，不铺垫废话
- 删除无效过渡句
- 快速切换感知焦点

## 四、减速技巧

- 插入环境或动作细节
- 给角色思考和误判空间
- 用沉默、停顿、空段制造延迟
- 用日常化细节接住高压剧情

## 五、章末钩子

常见有效方式：

- 抛出新问题
- 暗示新风险
- 让角色做出高代价决定
- 把冲突停在最关键一步
- 用一句反常识信息改变读者理解

不要每章都用同一种钩子。

## 六、视角切换

切换后第一段必须尽快说明：

- 现在是谁
- 现在在哪
- 现在的即时目标是什么

否则读者会先迷路，再出戏。

## 七、信息密度

| 场景类型 | 信息密度建议 |
|----------|--------------|
| 日常 | 低到中 |
| 设定说明 | 中到高 |
| 战斗 | 中 |
| 谈判/博弈 | 高 |
| 情感场景 | 低到中 |

世界观信息更适合渐进投喂：

1. 先出现术语
2. 再给上下文暗示
3. 再在需要时正式讲解

## 八、自检

- [ ] 本章有没有节奏变化
- [ ] 是否存在连续大段信息倾倒
- [ ] 高强度段落后是否给了消化空间
- [ ] 章末是否有前进力
