# 世界观核心规则

## 一、双界结构

### 现实之形 vs 灵界之义

- **现实之形**：能被物质、距离、能量和工程约束锁定的外在存在
- **灵界之义**：某一存在在灵界中被梦、叙事、信仰和集体记忆承认的意义结构

### 灵界三层

| 层级 | 名称 | 特征 | 对应修行 |
|------|------|------|----------|
| 近岸 | 梦潮带 | 最接近现实梦境与短期情绪 | 点火、低位铭印 |
| 中层 | 叙信海 | 角色、信仰、社会叙事 | 深位铭印、授箓、显相 |
| 深层 | 本相渊 | 古老原型、神性生物 | 开域、留痕、飞升 |

---

## 二、折熵天梯

来自第零文明——折熵文明的超早期飞升遗产，把"无序毁灭"改造成了"有序试炼"。

### 四个功能

- **折叠**：把随机熵灾整理成可识别的灾难类型
- **分流**：把灾难分配到不同层级
- **对位**：让挑战总有承载者与解法
- **留痕**：把通过与失败刻进世界

## 五、校勘权限

主角的特殊权限，能对回响中的历史分支进行局部改写。

### 核心规则

1. 回响里的事件都可以改（有权限前提下）
2. 改动只吃权限，不额外付代价
3. 改动会真实反馈到现世
4. 但回响外的大历史会往原来的毁灭线收束

### 历史收束

不是把改动抹掉，而是把"本来不该发生的改变"重新整理成"仿佛一直如此"的版本。

## 六、四名位系统

主角的特殊能力：在同一条真实本相上，开出四个可切换、可被承认的名位接口。

### 核心规则

1. **记忆实时同步**：四个名位共用同一套记忆
2. **不能并行现身**：同一时刻只能前显一个名位
3. **一位一路**：每个名位只走自己对应的主路
4. **名位会被禁用**：后期可能因高位打击而失去某个名位

### 四名位

| 名位 | 对应主路 | 特点 |
|------|----------|------|
| 命炉位 | 命炉道 | 感知生命能量 |
| 契律位 | 契律道 | 感知规则约束 |
| 构装位 | 构装道 | 感知结构完整性 |
| 叙演位 | 叙演道 | 感知叙事张力 |

---

## 七、神位政治法则

### 单锚法则

正位、对位、同类概念偏移的锚位神明，**同一时间只能存在一个**。

### 剧情冲突类型

- **继位**：继承已逝神明的锚位
- **弑神夺位**：斩杀现任锚主，夺取神性
- **扭位**：将神性轴扭向不同方向
- **偏位**：在神性轴上偏向一侧，形成子轴
- **竞锚**：多个存在争夺同一锚位

## 八、关键概念速查

| 概念 | 解释 |
|------|------|
| **印** | 你做过什么、成为什么、被什么记住 |
| **箓** | 这些印被什么接口重新组织并合法化 |
| **相** | 被组织起来的你，以何种外观向外显形 |
| **域** | 你的相足够稳定后，周围世界短时按你的逻辑运转 |
| **痕** | 你的域与事业长期留下来的不可逆刻印 |
| **锚** | 最终沉入本相渊让世界记住你的事物 |
| **入戏** | 被叙事结构拖进去，按桥段逻辑行动 |
| **出戏** | 主动打破角色逻辑，回归自我 |
