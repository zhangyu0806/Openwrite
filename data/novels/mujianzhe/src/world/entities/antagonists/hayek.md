# 哈耶克（未竟之市神）

## 基本属性

| 属性 | 描述 |
|------|------|
| **类型** | 偏移篡位型半神 |
| **所属时代** | 前文明末期埋线，帝国时期全面成敌 |
| **对应轴** | 远潮的"开拓未知与交通"被偏移成"探索市场与航路贸易" |

## 背景

普鲁托斯联邦与深井财团那边，本来就有一条线在计划攫取寰宇远潮盟背后代表探索/交通的神性，去造一个贸易与市场之神"哈耶克"。

它不是从"远航"的对位负面神性里长出来的，而是把同一条轴硬生生掰歪：

| 原意 | 偏移后 |
|------|--------|
| 开拓未知的探索精神 | 寻找可变现的新市场 |
| 交通连接 | 贸易航路 |
| 冒险归来 | 盈利回报 |
| 灯塔指引 | 价格指数指引 |
| 未知的彼岸 | 可交易的资源地 |

于是它变成：**未竟之市神·哈耶克**。

它现在还没完全成锚位，所以是半神。但一旦它成功，远潮那条轴就会整体发生偏移——正神不会和它并坐，只会被打落成残神、链位，或被迫退成旧时代幸存体。

## 可怕之处

它不是让世界马上毁掉，而是让世界越来越"高效贸易"，同时失去探索未知的动力。最后会出现：
- 航路不再通向未知，只通向可交易的市场
- 人的冒险精神变成投资回报率计算
- 城与城之间的关系先被贸易化，再被金融化
- 连神灵复苏资格都能被包装成可投资的期权

## 目标

不是狠狠干死帝国，而是篡改帝国对"探索 / 开拓 / 远行"的理解本身——让"走出去"不再是为了发现未知，而是为了开辟新市场。

这类敌人特别高级，因为它很多时候真的能让一部分人过得更好，但代价是把整条神性轴都换了芯。

## 适合的剧情

- 镜潮 / 远岸 / 星枢三域联动大卷
- 帝国和联邦的制度战
- 主角救一位"值得归来"的旧神时，被哈耶克体系强行包装成可交易项目
