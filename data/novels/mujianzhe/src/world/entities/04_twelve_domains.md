# 十二冠域总览

## 冠域本质

冠域不只是行政区，也是某种处理特种职能的机构，有些还会专门化负责一些特殊种类的回响。

帝国所谓"冠域"，并不首先是地理概念，而是**长期处理某一类双界压力的专业结构**。

一个地方之所以能成为冠域，不在于疆域大小，而在于它能否长期成立以下三件事：

1. 有稳定承接某类回响或者完成某些职责的科技
2. 能承担作为一个行政区的能力
3. 有可持续供给的授箓接口与职业路径

所以冠域的本质，不是"地方特色"，而是"帝国把哪类压力交给这里长期吞下"。

---

## 十二冠域的共同处理链

所有冠域都服从同一条基本逻辑：

1. **本域耗散**：先用本域最擅长的制度和行业吃掉第一波压力
2. **跨域转相**：本域吃不下的残响，被改写成别域更容易承接的形式
3. **终端归井**：仍无法消化的活核，再送去高保密链或第零封井

因此，冠域不是把回响"关起来"，而是把回响持续改写成社会可承受的形态。

## 冠域的本质结论

把十二冠域压缩成一句话：

| 冠域 | 本质 |
|------|------|
| 黄炉 | 把战争做成军工 |
| 镜潮 | 把欲望做成市场 |
| 白园 | 把畸变做成医疗 |
| 赤律 | 把失序做成法律 |
| 星枢 | 把时间偏差做成历法 |
| 文衡 | 把认知污染做成教育与校勘 |
| 归墟 | 把历史残渣做成回收与封存 |
| 远岸 | 把边界侵蚀做成边防与归航 |
| 青垣 | 把饥荒做成农时与储备 |
| 渊井 | 把灵界潮能做成能源 |
| 幽冥 | 把承认危机做成祭礼与册封 |
| 万幕 | 把群体情绪做成娱乐、传媒与梦网 |

所谓冠域，就是帝国把回响技术化、行业化、制度化之后留下来的十二种长期处理面。

## 十二冠域之间的分工关系

如果把十二冠域放在同一张图里看，大致可以分成四类：

### 一、正面承压域
黄炉、白园、青垣、远岸——直接面对战争、疾病、饥荒、边界接触等第一线压力。

### 二、秩序编译域
赤律、文衡、星枢、幽冥——负责把已经发生的压力重新编进法、史、时、礼四种可长期维持的秩序里。

### 三、资源与流通域
镜潮、渊井——一个处理价值流，一个处理能量流，都是帝国的基础供血系统。

### 四、吸渣与泄洪域
归墟、万幕——一个吞历史残渣，一个吞情绪与叙事余波，都是社会稳定的末端缓冲层。

这也是为什么十二冠域缺一不可：它们不是十二种风格，而是十二种必须长期有人承担的代价。
