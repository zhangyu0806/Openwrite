# 反派与灾难设计

## 文档定位

这份不是终稿，也不是要立刻全部写进正文的总设定。它更像一份后备草稿，用来先把未来可能会用到的反派、灾难、敌对路线与卷级冲突类型分好类，后面写到哪一卷，再从里面抽取、细化、组合。

原则上，这里只搭骨架，不把细节一次写死。

---

## 使用原则

未来的敌人最好不要全都是"纯坏人"。

这个世界真正有意思的冲突，至少应该分成四类：

1. **单纯会把世界推坏的东西**
2. **也想帝国更好、但走的是另一条路的人**
3. **借旧神、旧技术、旧历史做坏事的组织**
4. **帝国内部那些有能力、有野心、也有自己逻辑的人**

只要这四类都在，故事就不会只剩下"打怪升级"。

## 神位政治法则

### 1. 单锚法则

正位、对位、以及同类概念偏移出来的锚位神明，**同一时间只能存在一个**。

也就是说，这些如果被认定仍在同一条神性轴上，就不能并列坐稳两个锚位：
- "生机 / 灾疫"
- "远航 / 市场化航路"
- "故事 / 唯一主角"
- "轮转 / 不肯终结"

所以真正的剧情冲突天然就变成：
- **继位**——继承已逝神明的锚位
- **弑神夺位**——斩杀现任锚主，夺取神性
- **扭位**——将神性轴扭向不同方向
- **偏位**——在神性轴上偏向一侧，形成新的子轴
- **竞锚**——多个存在争夺同一锚位的所有权

而不是"两个神互相站着对波"。

### 2. 神性偏移法则

神性不是死的。同一条轴会随着文明使用方式、信徒理解、掌位者性格、历史压力发生偏移与扭转。

例如：
- 生机可以偏成**畸生**
- 远航可以偏成**交易与追索**
- 故事可以偏成**角色暴政**
- 轮转可以偏成**不死不活的滞留**

所以"敌人"不一定来自另一条轴，也可能来自本轴被拧歪之后的新版本。

### 3. 对位负面神性的四种状态

**第一种：暂灭型**
被打碎、被杀掉、被驱散，但没有真正从世界里消失。一旦条件合适，会重新凝回来。

**第二种：尽掌型**
被正位神明彻底夺取、压住、编进自己体内。正常情况下不能独立存在，但可以被外力强行"撬出反面"。

**第三种：未尽掌握型**
表面上被压制，实际上没有压干净。于是正神本身就可能带着危险的第二面。

**第四种：外持型**
负面神性没被正位完全拿下，反而落在某个半神生物、链位神灵、残神后裔手里。它们就会一直盯着锚位，等机会弑神飞升。

### 4. 中立概念例外

有些概念性的神性，不适合硬做对位：
- 纯时间
- 纯历史校勘
- 纯计量
- 某些极抽象的观测性神性

这些更适合写成：
- 危险中性
- 未稳定
- 部分掌控
- 一旦被偏移就会出大事

而不是一定要强行配个"负面神"。

这样会比"全世界每条轴都必须有正邪双神"高级很多。

## 反派类型分类

### 一、持续性的灾难背景

| 类型 | 说明 |
|------|------|
| **熵灾残留** | 滞留型、侵蚀型、借壳型 |
| **错误复苏体** | 悲剧感、错位感 |
| **邪化低维投影** | 深层回流 |
| **失控回响** | 反复覆盖现实 |

### 二、异路救世派

| 派别 | 核心想法 | 与主角冲突 |
|------|----------|------------|
| **断灵派** | 切断两界联系 | 主角要复苏神灵 |
| **锁史派** | 禁止校勘历史 | 主角有校勘权 |
| **再启派** | 准备下一次重启 | 主角想保住当下 |
| **纯人派** | 不依赖旧神遗产 | 主角带旧时代权限 |

### 三、邪教与邪神组织

| 类型 | 特征 |
|------|------|
| **无序之种** | 攻击身份本身，克四名位系统 |
| **借幕型组织** | 借壳表演，适合万幕线 |
| **深渊型教团** | 迎接沉睡的坏东西 |
| **伪礼型组织** | 盗用正统礼制 |

### 四、前文明遗产型反派

| 类型 | 特征 |
|------|------|
| **正统继承者** | 争解释权和正统 |
| **旧技军用派** | 把旧技术军事化 |
| **存续邦与遗民群** | 有资格的另类延续 |

### 五、帝国内部的敌人

| 类型 | 特征 |
|------|------|
| **冠域野心家** | 独占资源，扩张权限 |
| **极端工具化派** | 一切必须可控 |
| **极端保守派** | 害怕任何变化 |
| **篡位者与伪承认者** | 抢位而非救局 |

## 与"复苏好神灵"主线如何对接

主角后期的大方向：找回、拼回、复苏、保护那些还能站在这一边的神灵与旧存在。

### 每复苏一位好神灵，就唤醒一类对应敌人

- 守护型神灵 → 想把它工具化的人
- 记忆/历史型神灵 → 锁史派或伪正典体系
- 梦、幕、角色相关神灵 → 借幕型邪教或邪化投影
- 秩序、册封、送渡相关神灵 → 篡位者与伪礼组织

### 不是所有旧神都必须立刻站队

有些神灵复苏之后，也可以先观望、先试探、先拒绝。这样世界会更真实。

### 好神灵也未必完全无问题

"好"不等于"没有旧时代包袱"。有些旧神即使值得救回来，也仍然带着老时代的视角、偏见、制度惯性和危险习惯。这反而更有戏。
