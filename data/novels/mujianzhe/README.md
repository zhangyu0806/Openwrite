# 《幕间编修者》—— OpenWrite 小说项目

## 项目信息

- **小说名称**：幕间编修者（又名：幕启之钥）
- **副标题**：归墟编修录
- **类型**：东方玄幻 + 科幻文明遗迹
- **预计篇幅**：第一卷 177章，约130万字

## 核心概念

> *"在既定剧本中寻找自由——当你被世界分配了一个角色，你是选择演好它，还是拒绝成为它？"*

主角沈烬作为穿越者，在折熵宇宙中以"幕间编修者"的身份，通过**四名位系统**在回响中校勘历史、在现世中守护文明。

## 世界观概要

这是一个**前文明飞升失败后的废墟时代**。

宇宙曾经历过辉煌的"飞升时代"，八大古文明各自发展出神明级的技术。但最终，一切都在"终末合灾"中崩溃，宇宙被迫"重启"。

如今是**重启后的帝国时代**。新的帝国在废墟上建立，试图用"十二冠域"的制度维持秩序。但前文明的遗产——技术、神灵、回响——仍在影响这个世界。

### 核心设定

- **折熵天梯**：第零文明留下的宇宙级制度，把无序熵灾改造成有序试炼
- **灵界三层**：梦潮带（近岸）、叙信海（中层）、本相渊（深层）
- **修行七阶**：点火→铭印→授箓→显相→开域→留痕→飞升
- **回响三类型**：自然回响、熵灾回响、史核回响
- **四名位系统**：命炉/契律/构装/叙演四路同修

## 第一卷概要

### 主题

**角色与自我的边界**——当你扮演一个角色太久，你还剩下多少真实的自己？

### 核心回响

**「观众席就位」**：百年前万戏社"角色位量产化"实验的失败现场，一百名试镜者全部"入戏过深"，成为回响中的"常驻演员"。

### 主要人物

| 角色 | 身份 | 特点 |
|------|------|------|
| **沈烬** | 幕间编修者（主角） | 穿越者，四名位系统持有者 |
| **裴织** | 构装师 | 现实线女主，黄炉旁支后裔，傲娇技术宅，与主角斗嘴日常 |
| **伶舟** | 见习替角 | 回响线女主，古灵精怪，不能离开回响 |
| **刑无咎** | 前赤律执法者 | mentor，旧案钩子 |
| **顾衡** | 档案仲裁员/队医 | 队友，白园背景，温柔大姐姐，负责医疗保障 |
| **燕回** | 外勤回收员 | 队友，沉默可靠 |
| **白续** | 高级审核官 | 第一卷反派，续存派成员 |

## 使用说明

### 启动 Goethe（规划模式）

```bash
openwrite goethe
```

用于：整理灵感、修人物、修设定、修大纲

### 启动 Dante（写作模式）

```bash
openwrite dante
```

用于：推进章节正文、审查、状态结算

### 查看状态

```bash
openwrite status
```

### 同步缓存

```bash
openwrite sync
```
